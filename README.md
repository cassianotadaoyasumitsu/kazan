## To start
Follow all steps to run the application locally.
To set up Basic Data, please run:
### Access the directory

```bash
cd kazan
```

### Install Bundle

```bash
Bundle install
```

### Install or upgrade Yarn

To install:
```bash
npm install --global yarn
```

or

To upgrade:
```bash
yarn upgrade
```

### Drop exist Database
```bash
rails db:drop
```

### Create a Database
```bash
rails db:create
```

### Migrate the Database
```bash
rails db:migrate
```

### Seed the Database
```bash
rails db:seed
```

### Start 
```bash
rails s
```

### The last result should show: "All Done", if you have any problem, feel free to ask

* rails db:drop (if needed)
